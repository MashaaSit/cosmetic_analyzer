# Cosmetic Acne Risk Analyzer

## О проекте
Rule-based система оценки риска акне на основе состава косметического средства

## Методология
- База из 200 ингредиентов с экспертной разметкой
- Каждому ингредиенту присвоены значения `acne_effect` и `allergen`
- Риск считается как взвешенное среднее с учётом позиции ингредиента в составе
- Покрытие базы: 54.8% 

## Ограничения
Неизвестные ингредиенты считаются нейтральными и гипоаллергенными.
База покрывает 54.8% реальных составов.

## Стек
